# File Ops

* how to build 
   <pre>run "go build -o fops github.com/kiedveian/JDExam/fops" </pre>

* Usage:
  *  fops [flags]
  *  fops [command]

* Available Commands
  * linecount Print line count of file
  * checksum  Print checksum of file
  * version     Show the version info
  * help         Help about commands
* Flags
  * -h, --help   help for fops

* known issues
